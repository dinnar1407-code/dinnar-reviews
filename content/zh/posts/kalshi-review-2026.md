## Kalshi 是什么？

Kalshi 是一家受美国商品期货交易委员会（CFTC）监管的衍生品交易所，你可以在上面买卖"事件合约"——也就是对真实世界结果的"是/否"二元合约。可以把它理解为合规版的 Polymarket，但有几个关键区别：

- **受 CFTC 监管**，这意味着完全符合美国法律。不需要 VPN，不需要加密货币钱包。
- **合约以美元结算。** 你通过 ACH 银行转账入金，交易，然后提现到同一银行账户。
- **每份合约都是二元的**：如果事件发生，"是"合约支付 $1.00；如果没发生，支付 $0.00。价格 = 市场隐含概率。
- **在所有50个州可用**，包括加州——这对任何形式的美国事件交易平台来说都非常罕见。

Kalshi 于2021年上线，目前已发展出 100+ 个市场，涵盖政治、加密货币价格、经济指标、气候记录、科学突破、娱乐和体育等类别。

### 网页端
网页界面干净、功能实用。主仪表板显示推荐市场、你的持仓以及可按类别筛选的市场列表。下单流程简单明了：选择方向（YES/NO），设定合约数量和限价，然后提交。

订单簿视图够用，但不如传统券商那么丰富。你可以看到买卖价差、最近成交和深度——足以做出明智的决策，但喜欢看市场数据的重度用户可能需要用 API。

### 移动端
Kalshi 的移动应用（iOS 和 Android）与网页端体验相当一致。适合查看持仓和快速下单。事件结算的推送通知系统很可靠——我参与的市场结算后几分钟内就收到了通知。

### API / kalshi-cli
这是对开发者和偏好自动化的交易者来说最有意思的部分。Kalshi 提供基于 RSA 密钥认证的 REST API。开源工具 `kalshi-cli` 封装了该 API，提供：

- 市场发现与筛选
- 下单、撤单和订单状态查询
- 持仓和余额跟踪
- 历史交易数据

认证需要生成 RSA 密钥对并将公钥注册到 Kalshi。这是一次性设置，大约需要10分钟。配置完成后，你可以编写交易策略脚本、构建仪表板，或将 Kalshi 数据集成到自己的工具中。

**需要注意：** API 速率限制合理但并不宽松。如果你打算运行高频策略，很快就会撞墙。这是面向零售用户的平台——不是为做市机器人设计的。

这是 Kalshi 最强的卖点之一：

| 费用类型 | 成本 |
|---|---|
| 交易手续费 | **$0**（大部分市场） |
| 入金（ACH） | **免费** |
| 出金（ACH） | **免费** |
| 闲置费 | **无** |
| 最低账户余额 | **无** |

你没看错：大部分市场零交易佣金。Kalshi 通过买卖价差赚钱——买家愿意付的价格和卖家要价之间的差额。对于高流动性市场（重大政治事件、热门加密货币价格目标），点差很窄——通常只有1-2美分。对于小众市场，点差可能更大，这实际上就是你的"交易成本"。

![Kalshi 费用表 — 大部分市场零手续费](/images/kalshi-fees-table.png)

**入金提醒：** 只接受美元 ACH 银行转账。不接受加密货币入金，也不能用信用卡。这既是优点（监管合规），也是缺点（比加密平台到账慢）。如果你习惯了 Polymarket 上的即时 USDC 入金，1-2天的 ACH 延迟会让你觉得慢。

## Kalshi 与竞品对比

### Kalshi vs Polymarket
这是所有人都在问的比较。

| 维度 | Kalshi | Polymarket |
|---|---|---|
| **监管** | CFTC 监管 | 无监管（加密货币） |
| **美国合法** | ✅ 全美50州 | ❌ 美国 IP 被封锁 |
| **入金方式** | 美元（ACH） | USDC（加密货币） |
| **结算** | 自动，现金结算 | 需手动申领 |
| **市场深度** | 中等 | 加密/政治市场更深 |
| **费用** | 无（仅点差） | 无 |
| **可用地区** | 全美50州（含加州） | 美国以外 |

不可否认，Polymarket 在加密货币和政治市场上流动性更深。但如果你在美国，Polymarket 被地理位置封锁，使用 VPN 访问会让你的资金面临风险。Kalshi 的 CFTC 监管意味着你的资金存放在受监管的隔离账户中。

### Kalshi vs PredictIt
PredictIt 是由惠灵顿维多利亚大学运营的学术预测市场，在 CFTC 的"不采取行动函"下运作——而非完整监管。

- **PredictIt 限制：** 每市场最多 $850 持仓，市场选择有限
- **PredictIt 费用：** 盈利部分收取10%，外加出金费
- **Kalshi 优势：** 无持仓限制（仅受余额约束），无盈利抽成

### Kalshi vs 传统体育博彩
DraftKings 和 FanDuel 对每笔投注收取 5-10% 的抽水。Kalshi 的零手续费、仅点差模式使其在交易而非赌博时从根本上更便宜。但 Kalshi 是一个交易所——你需要对手方来成交你的订单。而博彩平台始终自己充当交易对手。

## 优缺点

### ✅ 我喜欢的

1. **监管确定性。** 作为一个经历过加密货币交易所关闭和账户冻结的人，知道 Kalshi 受 CFTC 监管、客户资金被适当隔离，让人真的放心。

2. **零交易手续费。** 你保留100%的收益。点差是你唯一的成本，而且在主要市场上点差很窄。

3. **自动结算。** 市场根据官方数据源自动结算。你不需要"申领"任何东西——获胜合约以现金形式支付到你的余额。Polymarket 需要手动申领，这是一个非常奇怪的用户体验设计。

4. **多元化的市场类别。** 除了常见的政治和加密货币，Kalshi 还有关于气候记录、科学里程碑、娱乐发布和经济指标的市场。这不只是一个博彩网站——它是一个真正的事件交易平台。

5. **移动应用。** 功能齐全、界面干净、推送通知可用。不算革命性，但够用。

6. **API 访问。** REST API 和 kalshi-cli 工具使得将 Kalshi 集成到自动化工作流成为可能。对数据驱动型交易者来说，这是被低估的功能。

### ❌ 我不喜欢的

1. **交易时段外的流动性问题。** 这是最大的实际困难。非体育事件市场在美国交易时段（周一至周五，大约东部时间 9:30 AM–4 PM）最活跃。在这些时段之外——尤其是周末——非体育市场非常淡。你会看到很宽的点差、很少的挂单，以合理价格成交限价单可能需要几个小时。

2. **体育串关玩法主导信息流。** 主页市场信息流越来越多地被复杂的体育串关玩法挤占（某球员得分超 X + 某队获胜等组合）。如果你关注的是经济、政治或加密货币市场，就得在大量体育内容中筛选。这对商业未必是坏事（体育能带来交易量），但稀释了平台作为预测市场的身份。

3. **非体育市场的周末死区。** 周五晚上到周日，非体育市场基本暂停。做市商撤退，点差扩大，订单簿变薄。围绕交易时段来规划你的交易。

4. **仅支持 ACH 入金。** 我理解原因——监管要求。但1-2个工作日的 ACH 延迟与加密平台相比显得非常过时。如果你想冲进一个突发热点事件的交易，就需要平台上已经有资金。

5. **API 文档有缺口。** REST API 可以用，但官方文档有些地方比较简略。kalshi-cli 工具能帮你弥补差距，但你很可能需要阅读源码才能理解某些端点。

6. **短期市场有限。** 大多数市场的到期日是数周到数月后。如果你想要当天或次日的短期事件合约（比如"今天 X 会发生吗？"），选择有限。Kalshi 一直在增加短期市场，但这个品类仍然落后于 Polymarket。

## 适合谁用？

### ✅ 适合：
- **美国交易者**，希望合法、受监管的事件交易
- **宏观/经济爱好者**，想对经济指标表达观点
- **加密货币交易者**，想要受监管的价格目标交易
- **数据驱动型用户**，想要 API 访问预测市场数据
- **小资金交易者**——无最低限额、无手续费，$10 就能开始

### ❌ 不太适合：
- **非美国居民**——Polymarket 或本地替代品可能提供更多市场
- **高频交易者**——API 速率限制和交易时段约束策略
- **纯体育博彩玩家**——DraftKings/FanDuel 提供更好的流动性和即时成交
- **周末交易族**——如果你主要在周末交易，流动性问题会让你沮丧
