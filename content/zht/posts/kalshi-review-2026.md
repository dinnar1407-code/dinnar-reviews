## Kalshi 是什麼？

Kalshi 是一家受美國商品期貨交易委員會（CFTC）監管的衍生品交易所，你可以在上面買賣「事件合約」——也就是對真實世界結果的「是/否」二元合約。可以把它理解為合規版的 Polymarket，但有幾個關鍵區別：

- **受 CFTC 監管**，這意味著完全符合美國法律。不需要 VPN，不需要加密貨幣錢包。
- **合約以美元結算。** 你通過 ACH 銀行轉帳入金，交易，然後提現到同一銀行帳戶。
- **每份合約都是二元的**：如果事件發生，「是」合約支付 $1.00；如果沒發生，支付 $0.00。價格 = 市場隱含概率。
- **在所有50個州可用**，包括加州——這對任何形式的美國事件交易平台來說都非常罕見。

Kalshi 於2021年上線，目前已發展出 100+ 個市場，涵蓋政治、加密貨幣價格、經濟指標、氣候記錄、科學突破、娛樂和體育等類別。

## 我是怎麼測試的

我不會在沒投入真金白銀的情況下寫這篇評測。以下是我實際做的事情：

**開戶：** 我建立了帳戶，完成了 KYC 身份驗證（任何 CFTC 監管交易所的標準流程），並通過 Plaid 關聯了銀行帳戶以進行 ACH 入金。驗證過程大約5分鐘——比大多數加密貨幣交易所快。

**入金：** 我存入了 $10.02。沒打錯——十美元零兩分。我想用小資金帳戶的真實體驗來測試平台，這也是大多數人開始的方式。ACH 轉帳在2個工作日內到帳。

**交易活動：** 在兩週內，我跨多個市場下了單：
- 「比特幣在2026年5月會收於 $150,000 以上嗎？」——以 $0.88 買入 NO
- 「2026年6月聯邦基金利率目標區間？」——分散在多個結果上
- 幾個短期經濟指標市場

**API 與 CLI：** 我用 RSA 簽名 API 認證配置了 kalshi-cli。這個 CLI 支援訂單管理（下單、取消、查詢訂單）、市場數據查詢和持倉追蹤。它是可以用的——但文件有待改善。如果你習慣用命令列工具，這是繞過網頁 UI 直接與交易所互動的好方式。

**總計投入時間：** 大約8-10小時的活躍使用，涵蓋網頁端、行動端和 CLI。

### 網頁端
網頁介面乾淨、功能實用。主儀表板顯示推薦市場、你的持倉以及可按類別篩選的市場列表。下單流程簡單明瞭：選擇方向（YES/NO），設定合約數量和限價，然後提交。

訂單簿視圖夠用，但不如傳統券商那麼豐富。你可以看到買賣價差、最近成交和深度——足以做出明智的決策，但喜歡看市場數據的重度用戶可能需要用 API。

### 行動端
Kalshi 的行動應用（iOS 和 Android）與網頁端體驗相當一致。適合查看持倉和快速下單。事件結算的推播通知系統很可靠——我參與的市場結算後幾分鐘內就收到了通知。

### API / kalshi-cli
這是對開發者和偏好自動化的交易者來說最有趣的部分。Kalshi 提供基於 RSA 金鑰認證的 REST API。開源工具 `kalshi-cli` 封裝了該 API，提供：

- 市場發現與篩選
- 下單、撤單和訂單狀態查詢
- 持倉和餘額追蹤
- 歷史交易數據

認證需要生成 RSA 金鑰對並將公鑰註冊到 Kalshi。這是一次性設定，大約需要10分鐘。配置完成後，你可以編寫交易策略腳本、構建儀表板，或將 Kalshi 數據整合到自己的工具中。

**需要注意：** API 速率限制合理但並不寬鬆。如果你打算執行高頻策略，很快就會撞牆。這是面向零售用戶的平台——不是為做市機器人設計的。

## 定價與費用

這是 Kalshi 最強的賣點之一：

| 費用類型 | 成本 |
|---|---|
| 交易手續費 | **$0**（大部分市場） |
| 入金（ACH） | **免費** |
| 出金（ACH） | **免費** |
| 閒置費 | **無** |
| 最低帳戶餘額 | **無** |

你沒看錯：大部分市場零交易佣金。Kalshi 通過買賣價差賺錢——買家願意付的價格和賣家要價之間的差額。對於高流動性市場（重大政治事件、熱門加密貨幣價格目標），點差很窄——通常只有1-2美分。對於小眾市場，點差可能更大，這實際上就是你的「交易成本」。

![Kalshi 費用表 — 大部分市場零手續費](/images/kalshi-fees-table.png)

**入金提醒：** 只接受美元 ACH 銀行轉帳。不接受加密貨幣入金，也不能用信用卡。這既是優點（監管合規），也是缺點（比加密平台到帳慢）。如果你習慣了 Polymarket 上的即時 USDC 入金，1-2天的 ACH 延遲會讓你覺得慢。

## Kalshi 與競品對比

### Kalshi vs Polymarket
這是所有人都在問的比較。

| 維度 | Kalshi | Polymarket |
|---|---|---|
| **監管** | CFTC 監管 | 無監管（加密貨幣） |
| **美國合法** | ✅ 全美50州 | ❌ 美國 IP 被封鎖 |
| **入金方式** | 美元（ACH） | USDC（加密貨幣） |
| **結算** | 自動，現金結算 | 需手動申領 |
| **市場深度** | 中等 | 加密/政治市場更深 |
| **費用** | 無（僅點差） | 無 |
| **可用地區** | 全美50州（含加州） | 美國以外 |

不可否認，Polymarket 在加密貨幣和政治市場上流動性更深。但如果你在美國，Polymarket 被地理位置封鎖，使用 VPN 存取會讓你的資金面臨風險。Kalshi 的 CFTC 監管意味著你的資金存放在受監管的隔離帳戶中。

### Kalshi vs PredictIt
PredictIt 是由威靈頓維多利亞大學營運的學術預測市場，在 CFTC 的「不採取行動函」下運作——而非完整監管。

- **PredictIt 限制：** 每市場最多 $850 持倉，市場選擇有限
- **PredictIt 費用：** 盈利部分收取10%，外加出金費
- **Kalshi 優勢：** 無持倉限制（僅受餘額約束），無盈利抽成

### Kalshi vs 傳統體育博彩
DraftKings 和 FanDuel 對每筆投注收取 5-10% 的抽水。Kalshi 的零手續費、僅點差模式使其在交易而非賭博時從根本上更便宜。但 Kalshi 是一個交易所——你需要對手方來成交你的訂單。而博彩平台始終自己充當交易對手。

## 優缺點

### ✅ 我喜歡的

1. **監管確定性。** 作為一個經歷過加密貨幣交易所關閉和帳戶凍結的人，知道 Kalshi 受 CFTC 監管、客戶資金被適當隔離，讓人真的放心。

2. **零交易手續費。** 你保留100%的收益。點差是你唯一的成本，而且在主要市場上點差很窄。

3. **自動結算。** 市場根據官方數據源自動結算。你不需要「申領」任何東西——獲勝合約以現金形式支付到你的餘額。Polymarket 需要手動申領，這是一個非常奇怪的用戶體驗設計。

4. **多元化的市場類別。** 除了常見的政治和加密貨幣，Kalshi 還有關於氣候記錄、科學里程碑、娛樂發布和經濟指標的市場。這不只是一個博彩網站——它是一個真正的事件交易平台。

5. **行動應用。** 功能齊全、介面乾淨、推播通知可用。不算革命性，但夠用。

6. **API 存取。** REST API 和 kalshi-cli 工具使得將 Kalshi 整合到自動化工作流成為可能。對數據驅動型交易者來說，這是被低估的功能。

### ❌ 我不喜歡的

1. **交易時段外的流動性問題。** 這是最大的實際困難。非體育事件市場在美國交易時段（週一至週五，大約東部時間 9:30 AM–4 PM）最活躍。在這些時段之外——尤其是週末——非體育市場非常淡。你會看到很寬的點差、很少的掛單，以合理價格成交限價單可能需要幾個小時。

2. **體育串關玩法主導資訊流。** 主頁市場資訊流越來越多地被複雜的體育串關玩法擠佔（某球員得分超 X + 某隊獲勝等組合）。如果你關注的是經濟、政治或加密貨幣市場，就得在大量體育內容中篩選。這對商業未必是壞事（體育能帶來交易量），但稀釋了平台作為預測市場的身份。

3. **非體育市場的週末死區。** 週五晚上到週日，非體育市場基本暫停。做市商撤退，點差擴大，訂單簿變薄。圍繞交易時段來規劃你的交易。

4. **僅支援 ACH 入金。** 我理解原因——監管要求。但1-2個工作日的 ACH 延遲與加密平台相比顯得非常過時。如果你想衝進一個突發熱點事件的交易，就需要平台上已經有資金。

5. **API 文件有缺口。** REST API 可以用，但官方文件有些地方比較簡略。kalshi-cli 工具能幫你彌補差距，但你很可能需要閱讀原始碼才能理解某些端點。

6. **短期市場有限。** 大多數市場的到期日是數週到數月後。如果你想要當天或次日的短期事件合約（比如「今天 X 會發生嗎？」），選擇有限。Kalshi 一直在增加短期市場，但這個品類仍然落後於 Polymarket。

## 適合誰用？

### ✅ 適合：
- **美國交易者**，希望合法、受監管的事件交易
- **宏觀/經濟愛好者**，想對經濟指標表達觀點
- **加密貨幣交易者**，想要受監管的價格目標交易
- **數據驅動型用戶**，想要 API 存取預測市場數據
- **小資金交易者**——無最低限額、無手續費，$10 就能開始

### ❌ 不太適合：
- **非美國居民**——Polymarket 或本地替代品可能提供更多市場
- **高頻交易者**——API 速率限制和交易時段約束策略
- **純體育博彩玩家**——DraftKings/FanDuel 提供更好的流動性和即時成交
- **週末交易族**——如果你主要在週末交易，流動性問題會讓你沮喪

## 總結

Kalshi 是美國交易者能用到的最佳合規預測市場——實際上，它是唯一一個擁有完整 CFTC 監管、覆蓋全美50個州的平台。

對於 $10.02 的帳戶來說，這個平台的體驗完全符合預期。零手續費意味著小餘額不會被佣金吃掉。自動結算系統可靠。API 讓你在網頁介面之外有了更多選擇。

流動性狀況是真正的制約因素。如果你能在美國交易時段內交易，並專注於主要事件市場，你會體驗不錯。如果你在週六下午試圖交易小眾的氣候或科學市場，你就只能盯著空空如也的訂單簿發呆了。

**我的建議：** 從 $10–50 開始。交易幾個市場來理解事件合約的運作方式。如果你有技術背景，可以使用 API——這是平台最有趣的部分。並在交易時段內交易以獲得最佳成交。
